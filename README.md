# Running Tracker

Трекер беговых тренировок — план подготовки к полумарафону **09.08.2026**.

**Цель:** 21,1 км за 1:40 (темп 4:44/км)

## Функции

- **План** — 13-недельный календарный план с выделением текущей недели и стартов
- **Пробежка** — запись каждой тренировки (дистанция, темп, пульс, самочувствие)
- **Журнал** — хронология всех пробежек с цветовой индикацией темпа
- **Аналитика** — графики недельного объёма и динамики темпа
- **Корректировка** — автоматический анализ и рекомендации по плану
- **Калькулятор** — расчёт темпа по дистанции и времени, и наоборот

## Запуск

Сайт задеплоен через GitHub Pages:
**https://aabramov77.github.io/running-tracker_new/**

Для локального открытия: откройте `index.html` в браузере.

## Архитектура

```
GitHub Pages (index.html + app.js)
        ↓  REST API
Cloud Run Function (main.py)
        ↓
GCS Bucket: running-tracker-aabramov77/runs.json
```

Данные хранятся в Google Cloud Storage. При отсутствии связи приложение работает из кэша `localStorage`.
