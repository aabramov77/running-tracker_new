# Running Tracker 🏃

Трекер беговых тренировок — план подготовки к полумарафону **09.08.2026**.

**Цель:** 21,1 км за 1:40 (темп 4:44/км)

## Функции

- 📋 **План** — 13-недельный календарный план с выделением текущей недели и стартов
- ➕ **Пробежка** — запись каждой тренировки (дистанция, темп, пульс, самочувствие)
- 📓 **Журнал** — хронология всех пробежек с цветовой индикацией темпа
- 📊 **Аналитика** — графики недельного объёма и динамики темпа
- 🔧 **Корректировка** — автоматический анализ и рекомендации по плану

## Запуск

Откройте `index.html` в браузере или задеплойте через GitHub Pages.

### GitHub Pages

1. Settings → Pages
2. Source: Deploy from a branch → `main` / `root`
3. Сайт будет доступен по адресу `https://aabramov77.github.io/running-tracker_new/`
